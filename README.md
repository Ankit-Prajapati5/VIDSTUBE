# VIDSTUBE

VIDSTUBE is a simple React web application that provides quick access to download videos from popular social media platforms such as **YouTube, Instagram, and Facebook**.

The project is built using **React** and **Tailwind CSS** and focuses on creating a clean and responsive user interface.

## Features

* Responsive design (mobile and desktop friendly)
* Simple and clean user interface
* Quick access to download services
* Built with React and Tailwind CSS

## Technologies Used

* React.js
* Tailwind CSS
* React Icons

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/yourusername/VIDSTUBE.git
```

2. Go to the frontend folder

```
cd VIDSTUBE/frontend
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm run dev
```
